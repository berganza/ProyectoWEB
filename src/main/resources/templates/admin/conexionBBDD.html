Conexión básica:

Hay que tener en cuenta que la Base de datos debe existir en MySQL, es decir, primero hay que crear la base de datos para poder hacer la conexión. Con el nombre sería suficiente ya que las tablas serán generadas por la aplicación al momento de conectar.

spring.datasource.url = jdbc:mysql://localhost:3306/NOMBRE_DE_LA_BBDD
spring.datasource.username = SU_USERNAME
spring.datasource.password = SU_CONTRASEÑA
spring.datasource.driver-class-name = com.mysql.cj.jdbc.Driver

spring.jpa.show-sql = true
spring.jpa.open-in-view = false
spring.jpa.hibernate.ddl-auto = update
spring.jpa.properties.hibernate.dialect = org.hibernate.dialect.MySQL5InnoDBDialect
