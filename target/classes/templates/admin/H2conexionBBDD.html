En el archivo "application.properties" escribimos el siguiente ejemplo de conexión inicial:

# H2
spring.h2.console.enabled = true
spring.h2.console.path = /console
# Datasource
spring.datasource.url = jdbc:h2:mem:emdao;DB_CLOSE_DELAY=-1;DB_CLOSE_ON_EXIT=FALSE
spring.datasource.username = sa
spring.datasource.password =
spring.datasource.driver-class-name = org.h2.Driver

** Con la instrucción "spring.h2.console.path = /console" le estamos indicando que tenemos acceso a la base de datos en la dirección "http://localhost:8080/console" para poder gestionar directamente nuestras consultas en SQL.
